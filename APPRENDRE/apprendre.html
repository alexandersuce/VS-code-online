Documentation

HTML (HyperText Markup Language) est le code utilisé pour structurer une page web et son contenu. Voici un exemple de code HTML :

<!DOCTYPE html>
<html>
<head>
    <title>Titre de la page</title>
</head>
<body>
    <h1>Bonjour le monde</h1>
    <p>Ceci est un paragraphe.</p>
</body>
</html>

CSS (Cascading Style Sheets) est utilisé pour styliser le contenu HTML. Voici un exemple de code CSS :

body {
    background-color: #f0f2f5;
    font-family: Arial, sans-serif;
}
h1 {
    color: #333;
}
p {
    color: #666;
}

JavaScript est utilisé pour ajouter de l'interactivité aux pages web. Voici un exemple de code JavaScript :

document.querySelector('h1').addEventListener('click', function() {
    alert('Titre cliqué!');
});
